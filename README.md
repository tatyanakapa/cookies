# 🍪 Cookies Management System

## 📌 Описание
Проект реализует систему управления cookies в веб-приложении для работы с пользовательскими сессиями и данными.

## 🎯 Цель
Изучение механизмов хранения данных пользователя и основ безопасности веб-приложений.

## ⚙️ Технологии
- HTML
- JavaScript
- Git

## 🚀 Функционал
- Работа с cookies
- Хранение пользовательских данных
- Обработка данных на стороне клиента

## 🔐 Безопасность
- HttpOnly / Secure cookies (планируется)
- Защита пользовательских данных

## 📈 Планы развития
- Добавление авторизации
- Интеграция с backend
- Использование AI для анализа пользовательского поведения
